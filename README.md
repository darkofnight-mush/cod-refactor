This plugin provides a registry-driven foundation for Chronicles of Darkness
games running on AresMUSH 2.8+.

Included:
- Character Type registry
- State registry
- Stat registry (Attributes, Skills, Merits, Demon stats)
- Refactored `sheet` and `sheet/set` commands

Excluded (by design):
- Chargen UI changes
- Approval workflow
- Ember changes
- Power automation
